# Forum Vacuun

> Frontend do projeto

## Tecnologias

- Next Framework - v13.2.4
- Nodejs - v19.8.1

## Como rodar o projeto no ambiente de desenvolvimento

Clone o projeto

```bash
git clone git@github.com:VacuumORG/forum_vacuun_front.git
```

Baixe as dependencias do projeto

```bash
npm i
# or
yarn i
# or
pnpm i
```

Comando para iniciar o projeto

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
